2. outline

左のデータで，

  1. linesを回しながら，
  2. key('* fix ',' 1 F')でstate(:wait_next, :searching)を変え,
  3. action(:xy_data, :z_data, :idle)をとる

LL.4-13で用意したHash構造を使って， LL.20-1で，line[0..5]をkeyとして[state, action]を調整している．．

3. description

詳しい解説．．．
